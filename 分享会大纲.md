## 介绍

分享会将以 群视频 + VSCode Live Share 的形式

内容是关于我最近学习 Vue 课程和资料的总结

主要是借助讲解 Vue 和大家交流 `框架学习的路径` 和 `怎么舒服的写框架`

所以不需要大家对 Vue 本身有所了解

本次分享会共两期内容，分为 基础原理 和 实践 两部分:

- 基础原理篇 (第一期)
    围绕 `Vue 的响应式` 和 `render 函数` 这两个核心部分，
    以 `编写 Vue 的插件` 的结尾

- 实践篇 (第二期)
    我们将使用 Vue 的一些基本功能从头开始构建简单版本的功能库。
    例如路由，状态管理，表单验证...

分享会除了讲解内容的同时，会有类似于上课材料的 git 仓库地址供大家下载，尽请期待 ^__^

希望能帮助你理解 Vue 响应式构建的原理，从而更好地利用 Vue 写出优雅的解决方案

        - 例子: 基于 prop 渲染一个元素
        - 例子: 基于 prop 动态渲染组件
        - 例子: 封装高阶组件

- 构建简单版本的功能库 - 第二期 (2小时)
    - 状态管理: 45分钟
